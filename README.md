Makina
------

Makina is unofficial Skype API for Windows. It implements by UI Automation that uses image recognition (OpenCV) and optical character recognition (Tesseract OCR).


## Development environments

  - JRuby
  - AutoHotKey


## System requirements

  - Windows 8.1
  - Skype 7.3
  - Java 1.8
  - JRuby 1.7
